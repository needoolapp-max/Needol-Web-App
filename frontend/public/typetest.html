Typing Test

This page has no JavaScript at all — no React, no scripts of any kind. If typing into the fields below freezes the page, the cause is in your browser, OS, or a browser extension — nothing we ship from the app can fix it. If typing here is smooth but typing on /login or /signup freezes, then it's something in our React code path.

1. Plain text no autocomplete

2. Email field native autofill

3. Password field password-manager target

4. Password field extension opt-out

How to test on each affected device:
  • Tap each field, type 10–20 characters, observe.
  • If none freeze → cause is in our React code path.
  • If only password (#3) freezes but opt-out password (#4) doesn't → a password manager or AV extension is the cause; our existing opt-out attributes work; the broken extension just needs disabling.
  • If all freeze → the cause is your browser or OS itself (iOS Safari autofill bug, broken extension that ignores opt-outs, hardware/VM issue). No code change to our app can fix it.
